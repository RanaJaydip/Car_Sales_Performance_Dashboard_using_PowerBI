# Car_Sales_Performance_Dashboard_using_PowerBI
🚗 Car Sales Performance Dashboard (Power BI)  This project is developed using Power BI as part of my Semester 7 Data Science coursework. The dashboard provides insights into car sales performance, customer trends, and revenue analysis to support business decision-making.<br>
🔍 Objectives

1.Analyze total car sales and revenue

2.Identify top-performing car brands and models

3.Understand customer purchasing behavior

4.Track sales trends over time

5.Compare regional performance
